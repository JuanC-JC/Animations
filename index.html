Documentacion Transformaciones y animaciones

Transformaciones Directas:

.translate{
transition: transform 0.5s , background-color 1s;
} .translateX:hover{
transform : translateX(100px);
background: yellow;
}

Son animaciones que ocurren por el cambio de propiedad, si el trigger es un hover no sera posible conservar el estado final de la animacion cuando se retire el hover del elemento, su ventaja es que cada propiedad puede tener una transicion diferente ejemplo la velocidad de translate 0.5s y el cambio de color 1s

Transformaciones con animaciones (css-nativo):

.translateX:hover{
animation: name-animation 1s;
}
Ventajas
  • Ajustar cada frame especifico
  • loops css nativo
  • trigger no dependiente del elemento
Desventajas
  • Tiempo de transicion igual (hay que calcular segun los frames)
  • Escribir retorno de la animacion
  • Sostener el hover para no cancelar la animacion
  • Casi imposible mantener animacion activa despues de retirar el hover
